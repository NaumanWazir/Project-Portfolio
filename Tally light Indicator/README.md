# 🎥 Tally Light Indicator

An **NDA project** developed for a **UK-based company**, involving the design of both **PCB hardware** and **firmware** for a **wireless tally light system** used in professional video studios.  
The system employs **PIC16 microcontrollers** and **easyRadio eRA900TRS** modules for reliable wireless communication between a central transmitter and multiple receivers.

---

## 🎯 Project Objectives
- Develop a **wireless tally light system** for broadcast studios to indicate which camera is live.  
- Ensure **synchronized communication** between the transmitter and multiple receivers.  
- Support both **tally control** and **iris adjustment** over a robust wireless link.  
- Create a **compact, production-ready PCB design** suitable for studio integration.  

---

## 👨‍💻 My Contributions
- Designed and implemented **firmware** for both **transmitter** and **receiver** units using **PIC16 microcontrollers**.  
- Developed a **wireless communication protocol** using **easyRadio eRA900TRS modules** for reliable multi-node control.  
- Designed **custom PCBs** for transmitter and receiver units in **EasyEDA**, optimized for compact form factor.  
- Simulated and verified system behavior in **Proteus** before hardware testing.  

---

## 🛠️ Technologies and Tools Used
- **Microcontroller:** PIC16 series  
- **Wireless Module:** easyRadio eRA900TRS  
- **Communication Type:** UART, Wireless  
- **Firmware Development:** MPLAB IDE (C language)  
- **Simulation Tool:** Proteus Design Suite
