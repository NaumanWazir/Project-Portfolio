# Prototype Testing for CurveAssure

This project was completed under a **Non-Disclosure Agreement (NDA)** with **[CurveAssure](https://www.curveassure.com/)**.  
CurveAssure provides an innovative, **non-invasive at-home spinal assessment system** designed for long-term biomechanical monitoring of the spine.

---

## 🧩 Project Objectives
- Develop firmware for the **CurveAssure prototype** based on the **nRF52840 microcontroller**.  
- Enable **reliable, lossless wireless data transmission** of biomechanical data over BLE.  
- Implement **low-power operation** suitable for wearable applications.  
- Support **data logging** to SD card for offline analysis.  
- Provide a **PC application** to process and analyze recorded motion data.

---

## 👨‍💻 My Contributions
- Designed and implemented the **embedded firmware** for sensor data acquisition, BLE communication, and SD card logging.  
- Integrated a **9-DOF IMU** for real-time motion capture (accelerometer, gyroscope, magnetometer).  
- Implemented **low-power sleep modes** and optimized timing for efficient wearable operation.  
- Developed a **Visual Studio desktop application** to process and visualize logged data.  
- Performed **prototype testing and calibration** to ensure data accuracy and transmission reliability.

---

## 🛠️ Technologies and Tools Used
- **Microcontroller:** Nordic nRF52840  
- **Sensors:** 9-DOF IMU (Accelerometer, Gyroscope, Magnetometer)  
- **Communication:** Bluetooth Low Energy (BLE)  
- **Storage:** SD Card (SPI Interface)  
- **Development Tools:** Visual Studio, Arduino IDE  
- **Programming Language:** C# / C++
