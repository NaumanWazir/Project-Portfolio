# 👟 Footpod Firmware Development

## 🎯 Project Objective  
The **Footpod Firmware** was developed for a wearable motion-tracking device designed to measure **distance** and **cadence** for athletes.  
The goal was to create **low-power, real-time firmware** capable of accurately processing motion data and providing instant performance feedback during training and recovery.  
*(Source code not shared due to NDA agreement.)*

---

## 💡 My Contributions  
- Designed and developed the **firmware architecture** for motion tracking and cadence computation.  
- Integrated the **Bosch BMI270 IMU** and **HJ840 MCU** for sensor fusion and wireless data communication.  
- Utilized the **Bosch Sensortec API** for real-time motion processing, calibration, and filtering.  
- Enabled **real-time alert functionality** to notify users when cadence or speed thresholds were exceeded.  

---

## 🧰 Technologies & Tools Used  
- **Microcontroller:** HJ840 (BLE-capable MCU)  
- **IMU Sensor:** Bosch BMI270  
- **API Library:** Bosch Sensortec API  
- **IDE:** SEGGER Embedded Studio  
- **Programming Language:** Embedded C  
- **Firmware Design:** Sensor fusion and low-power state management
