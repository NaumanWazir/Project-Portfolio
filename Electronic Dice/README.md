# 🎲 Electronic Dice

## 🎯 Project Objective  
The **Electronic Dice** project was created as a simple yet engaging demonstration of **embedded random number generation** and **digital display control** using a **PIC16F microcontroller**.  
The objective was to design a low-cost, fully functional **electronic dice** that mimics real dice rolling through electronic displays and push-button inputs.

---

## 💡 My Contributions  
- Designed the **firmware** to generate random numbers (0–6) for each 7-segment display.  
- Implemented **sequential button logic** for rolling, resetting, and powering down the displays.  
- Developed and tested the **Proteus simulation** to verify timing, display output, and user interaction.   

---

## 🧰 Technologies & Tools Used  
- **Microcontroller:** PIC16F series  
- **Displays:** Dual 7-segment displays with decoders  
- **Input Devices:** Push buttons for control and reset  
- **IDE:** MPLAB X  
- **Simulation Tool:** Proteus Design Suite  
- **Programming Language:** Embedded C
