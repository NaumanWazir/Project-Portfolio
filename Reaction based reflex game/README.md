# 🎮 Reflex Button Game System

This project implements a **two-player reflex-based game** using the **MSP430G2553 microcontroller**.  
Players compete by reacting to LED color prompts, with the system determining the winner based on **reaction time** and penalizing incorrect responses.  
The program was developed in **Code Composer Studio**, utilizing **interrupt-driven inputs** and **hardware timers** for precise response detection.

---

## 🧩 Project Objectives
- Design and implement a **two-player reflex competition system** using MSP430.  
- Measure and compare **reaction times** for each player based on LED prompts.  
- Implement **interrupt-driven button handling** for accurate timing.  
- Display **round results**, **time differences**, and **match winners** on a LCD display.  
- Automate game control with random LED prompts and timed restarts.

## 🛠️ Technologies and Tools Used
- **Microcontroller:** Texas Instruments MSP430G2553  
- **IDE:** Code Composer Studio  
- **Programming Language:** C  
- **Inputs:** 2 buttons per player (Red & Green)  
- **Outputs:** Red and Green LEDs, LCD Display  
- **Timing Mechanism:** Hardware Timers  
- **Interrupts:** Button Press and Timing Events  
- **Simulation Tool:** Proteus
