# 🔥 BBQ Blower Firmware

## 🎯 Project Objective  
The **BBQ Blower Firmware** was developed for a European client to control airflow in an **electric BBQ blower system**.  
The objective was to design a **stable, responsive, and efficient embedded control firmware** that manages motor speed, monitors system inputs, and provides clear user feedback through LEDs and sound alerts.

---

## 💡 My Contributions  
- Designed and developed the **firmware architecture** using a **state machine** approach for modular and predictable operation.  
- Implemented **PWM control** for precise motor speed and LED intensity management.  
- Integrated **ADC measurement** for real-time battery voltage monitoring.  
- Developed **button interrupt logic** for multi-function control using a single button based on press duration.  
- Added **buzzer and LED indicators** to provide intuitive user feedback.  
- Simulated and validated the complete firmware using **Proteus Design Suite** before hardware deployment.  

---

## 🧰 Technologies & Tools Used  
- **Microcontroller:** PIC16 series  
- **IDE:** MPLAB X  
- **Simulation Tool:** Proteus Design Suite  
- **Programming Language:** Embedded C  
- **Firmware Design:** State machine-based architecture
