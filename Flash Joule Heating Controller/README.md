# ⚡ Flash Joule Heating Controller

## 🎯 Project Objective  
The **Flash Joule Heating (FJH) Controller** was developed for a **U.S. client** to safely manage the **charging and discharging of high-energy capacitor banks** used in Flash Joule Heating applications.  
The main objective was to design **firmware that ensures precise sequencing, user safety, and real-time system monitoring** through a clear LCD interface.

---

## 💡 My Contributions  
- Designed and implemented a **4-stage state machine** for controlled capacitor charging, discharging, and system safety.  
- Developed firmware logic for **button input handling** and **digital output control** across 6 channels.  
- Integrated an **I²C-based LCD display** to provide real-time feedback on operational states.  
- Simulated and verified the complete control flow in **Proteus Design Suite** before hardware testing.  
- Delivered a **robust and fail-safe firmware solution** tailored for high-energy electrical systems.  

---

## 🧰 Technologies & Tools Used  
- **Microcontroller Platform:** Arduino-compatible MCU  
- **IDE:** Arduino IDE  
- **Display Interface:** I²C LCD module  
- **Simulation Tool:** Proteus Design Suite  
- **Firmware Design:** 4-state finite state machine  
- **Programming Language:** Embedded C / Arduino Framework
