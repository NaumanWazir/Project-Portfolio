# FOG-Based Inertial Navigation System (INS) for Sub-Orbital Flight 

This project explored the feasibility of using **Fiber Optic Gyroscopes (FOGs)** as an alternative to **Dynamic Tuned Gyroscopes (DTGs)** for inertial navigation.  

## Background  

By 2020, our team sought to validate whether **FOG-based IMUs** could provide the accuracy, reliability, and robustness required for aerospace applications. The goal was to:  
- Convert a **FOG-based IMU** into a fully functional **Inertial Navigation System**.  
- Test its performance in a **sub-orbital flight environment**.  
- Verify the accuracy of developed **navigation schemes** and **On-Board Computer (OBC) functionality** under real flight conditions.

## Outcome  

- Delivered a **flight-ready OBC PCB design**, enabling the first integration of a FOG-based INS within our program.  
- Successfully tested the system for **sub-orbital flight**, validating both the feasibility of FOG gyroscopes and the performance of navigation schemes.  
- Established a new pathway for **next-generation navigation systems** beyond DTG-based solutions.
