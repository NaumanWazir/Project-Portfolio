# 💡 Hex LED Intensity Controller

## 🎯 Project Objective  
The **Hex LED Intensity Controller** was developed for a **client in India** to provide **digital brightness control** for a dual-zone LED array.  
The objective was to create an **intuitive, reliable, and programmable LED control system** capable of storing presets and maintaining settings across power cycles.

---

## 💡 My Contributions  
- Designed and implemented the **firmware** for dual LED driver ICs to enable synchronized and independent brightness control.  
- Developed **button handling logic** with **software debouncing**, **short/long press detection**, and **preset memory functions**.  
- Integrated **EEPROM storage** to retain brightness settings and presets after power loss.  
- Simulated the full system in **Proteus** and later verified operation on **Wokwi** for real-time validation.  
- Delivered a **robust and portable firmware solution** easily adaptable to other Arduino-compatible microcontrollers.  

---

## 🧰 Technologies & Tools Used  
- **Microcontroller Platform:** Arduino-compatible MCU  
- **LED Drivers:** Dual driver ICs for independent control  
- **User Interface:** 8 push-buttons with software debouncing and multiplexing 
- **Non-Volatile Memory:** EEPROM for storing presets  
- **IDE:** Arduino IDE  
- **Simulation Tools:** Proteus Design Suite and Wokwi  
- **Programming Language:** Embedded C / Arduino Framework
