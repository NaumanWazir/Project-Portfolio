# 🎵 Sound Generator Toy

A **motion-activated sound generator prototype** developed for a **client in the USA**.  
The system uses a **PIC16F microcontroller** with a built-in **DAC** and an **MPU6050 accelerometer** to produce dynamic sound effects that change in response to motion and acceleration.

---

## 🎯 Project Objectives
- Develop a **motion-responsive sound generation system** for an interactive toy.  
- Use **accelerometer data** to modulate sound frequency in real time.   
- Achieve **smooth frequency transitions** based on the toy’s motion intensity.  

---

## 👨‍💻 My Contributions
- Designed and implemented **firmware in Embedded C** for the PIC16F microcontroller.  
- Integrated **MPU6050 accelerometer** via **I2C communication** for real-time motion sensing.  
- Developed a **dynamic DAC-based sound generation algorithm** driven by acceleration data. 
- Tested and optimized for **low latency and stable frequency output**.  

---

## 🛠️ Technologies and Tools Used
- **Microcontroller:** PIC16F (with onboard DAC)  
- **Sensor:** MPU6050 (3-axis accelerometer and gyroscope)  
- **Interface:** I2C communication  
- **IDE:** MPLAB X  
- **Programming Language:** Embedded C  
- **Output:** DAC audio signal to amplifier
