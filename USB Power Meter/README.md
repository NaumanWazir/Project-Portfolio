# ⚡ USB Power Meter

A compact and accurate **USB Power Meter** developed for a **UK-based client** to measure and visualize real-time voltage, current, and power consumption of USB-powered devices such as cameras, phones, and tablets.  
The device connects inline between the power source and load, displaying live data and graphical trends on an **ST7735S TFT display**.

---

## 🎯 Project Objectives
- Design a **real-time USB power monitoring system** for accurate voltage and current measurement.  
- Implement a **graphical user interface** to display live readings and power trends on a color TFT.  
- Ensure **high accuracy**, **compact design**, and **low power consumption** for portable testing.  
- Provide a reliable tool for monitoring USB-powered electronics during development and diagnostics.  

---

## 👨‍💻 My Contributions
- Developed complete **firmware in C** using **MPLAB X IDE** for the **PIC18 microcontroller**.  
- Implemented a **custom ST7735S TFT driver** for displaying voltage, current, and graphical waveforms.  
- Verified performance and accuracy using **Proteus simulation**.  
- Delivered a **fully functional and calibrated prototype** for field testing and production readiness.  

---

## 🛠️ Technologies and Tools Used
- **Microcontroller:** PIC18 series  
- **Display:** ST7735S TFT (Color Graphical Display)  
- **Measured Parameters:** USB Voltage, Current, and Power  
- **Development Environment:** MPLAB X IDE (XC8 Compiler)
