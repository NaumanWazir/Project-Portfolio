# GSM-Based Power Monitoring and Control Unit

## Overview
The project started in 2013, when the GPS development team was deploying their setup in remote areas. The problem was that equipment deployed required power monitoring and needed to be restarted if it hung. Due to the sensitive nature of the project, commercial IoT solutions could not be deployed. I was tasked to find out the solution.

## My Role
- Designed a custom system based on **SIM908 GSM module** and a microcontroller.  
- Implemented monitoring and control features for remote equipment.  

## Features
- Monitors **current, voltage, and temperature** of up to 10 devices.  
- User can:  
  - Request system status  
  - Set current/voltage thresholds (devices are cut off when exceeded)  
  - Enable **continuous report mode** (system sends status at a user-configurable fixed interval)  
  - Remotely **turn on/off each equipment**  

## Outcome
The system provided a reliable GSM-based power monitoring and control solution, ensuring remote equipment could be managed securely without relying on commercial IoT services.
