# ⚡ 6-Phase Automatic Change Over Controller

## 🎯 Project Objective  
The **6-Phase Automatic Change Over Controller** was developed for **industrial environments** requiring a **reliable and uninterrupted power supply**.  
Its main objective is to automatically monitor and switch between **main supply phases (R, S, T)** and **standby phases (U, V, W)** to ensure continuous operation during voltage fluctuations or power failures.

---

## 💡 My Contributions  
- Designed and implemented the **automatic phase detection and switching algorithm**.  
- Developed the **firmware** in Embedded C for **real-time voltage monitoring** and **relay control**.  
- Built and validated the **complete simulation** in Proteus Design Suite prior to hardware development.  
- Optimized **latching relay control logic** for smooth and failsafe phase transitions.
