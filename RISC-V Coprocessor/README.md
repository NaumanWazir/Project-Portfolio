# RISC-V Based Energy-Efficient Microcontroller with ECDSA Hardware Acceleration

## Overview
This project was my **Master’s Thesis in Integrated Circuit Design** at **TUM-Asia**, conducted in collaboration with **Infineon Technologies Singapore**.  
The goal was to design a **cost-effective, energy-efficient microcontroller** based on the **open-source RISC-V Instruction Set Architecture (ISA)**, tailored for embedded security applications.  

## Key Features
- **RISC-V ISA Core**: Open-source, customizable, and scalable for future extensions.  
- **ECDSA Hardware Acceleration**: Integrated accelerator optimized for **256-bit arithmetic operations**, enabling efficient elliptic curve cryptography.  
- **Performance Gains**: Hardware acceleration significantly reduced execution time compared to software-only implementations.  
- **Software Simplification**: Offloading intensive arithmetic operations to hardware simplified software routines and reduced code complexity.  

## Outcome
- Delivered a **secure, lightweight, and high-performance microcontroller** suitable for IoT and embedded cryptographic workloads.  
- Demonstrated the feasibility of using **RISC-V for energy-efficient, security-focused systems**.  
- Provided Infineon with a **validated proof-of-concept** for integrating ECDSA acceleration into RISC-V based designs.
